What is Node.js, why and when you should use it.

By Cyrus Khamak

2 years 2 month ago

Node.js is an open source, cross-platform built on Chromeâ€™s JavaScript runtime for fast and scalable server-side and networking applications.

That means it is fast, efficient and is a highly scalable web server for web applications.

Node.js is a good choice for applications that have to process a high volume of short messages requiring low latency. Such systems are called real-time applications (RTAs)

Node.js is especially suited for applications where you'd like to maintain a persistent connection from the browser back to the server. Using a technique known as "long-polling", you can write an application that sends updates to the user in real time!
